# scau-gis 🌍
华南农业大学 地理信息科学专业 专属学习&竞赛资料仓库，涵盖大一至大三全部核心课程资料、各类竞赛备赛素材，旨在为学弟学妹提供便捷的学习参考，助力专业学习与竞赛提升，打造专属华农GISer的资源聚集地。
## 仓库介绍
本仓库由华农GIS专业学生整理维护，内容涵盖课程课件、实验指导、期末复习笔记、历年真题、竞赛真题及备赛思路，所有资料均经过筛选，优先保证实用性和针对性，贴合华农GIS专业培养方案，适配本专业学弟学妹的学习需求。
## 使用说明
1. 资料获取：直接克隆本仓库至本地，或下载对应文件夹/文件即可使用；
2. 资料补充：若你有遗漏的课程资料、更优质的复习素材或竞赛经验，欢迎通过 Issue 提交，或发起 Pull Request 补充，共同完善仓库；
3. 资料纠错：若发现资料有误，请在 Issue 中说明，我会及时修正；
4. 建议反馈：关于仓库结构、资料分类的优化建议，也可通过 Issue 提出。
5. 也可以联系仓库管理员23地信F. Jason提出建议，他的联系方式是zhouzx55@163.com
## 声明
1. 本仓库所有资料均来自个人整理、老师课件分享及同学互助，仅用于学习交流，禁止用于商业用途、二次售卖或其他盈利行为；
2. 资料仅供参考，最终学习内容请以学校课程要求、老师授课内容为准；
