# OrbitVision Pro 打包说明

## 快速打包

### 方法一：使用打包脚本（推荐）
```powershell
python build_exe.py
```

### 方法二：直接使用PyInstaller命令
```powershell
pyinstaller --name="OrbitVision Pro" --onefile --windowed --add-data="nav;nav" --hidden-import=matplotlib --hidden-import=numpy --collect-all=matplotlib --noupx app/gui_gnss.py
```

## 打包参数说明

- `--name="OrbitVision Pro"` - 生成的exe文件名
- `--onefile` - 打包成单个exe文件（便于分发）
- `--windowed` - 无控制台窗口（GUI程序）
- `--add-data="nav;nav"` - 包含导航文件夹
- `--hidden-import=matplotlib` - 显式包含matplotlib
- `--collect-all=matplotlib` - 收集matplotlib所有依赖文件
- `--noupx` - 不使用UPX压缩（提高兼容性）

## 打包后文件位置

- 可执行文件：`dist/OrbitVision Pro.exe`
- 临时文件：`build/` 文件夹（可删除）
- 打包配置：`OrbitVision Pro.spec` 文件

## 运行打包后的程序

1. 进入 `dist` 文件夹
2. 双击 `OrbitVision Pro.exe` 运行
3. 首次运行可能需要几秒钟启动时间

## 分发说明

只需要分发 `dist/OrbitVision Pro.exe` 文件即可，用户无需安装Python环境。

## 常见问题

### 1. 打包后程序启动慢
- 这是正常现象，PyInstaller打包的程序首次启动需要解压
- 可以考虑使用 `--onedir` 模式替代 `--onefile`

### 2. 缺少matplotlib字体
- 打包时自动包含了matplotlib的字体文件
- 如果仍有问题，可以在打包脚本中添加字体路径

### 3. 程序体积较大
- matplotlib库较大（约100-150MB）
- 可以使用UPX压缩，但可能降低兼容性

### 4. 杀毒软件误报
- PyInstaller打包的程序有时会被误报
- 可以添加数字签名或提交白名单申请

## 高级选项

### 添加程序图标
1. 准备一个 `.ico` 格式的图标文件
2. 在打包命令中添加：`--icon=icon.ico`

### 打包成文件夹模式（启动更快）
```powershell
pyinstaller --name="OrbitVision Pro" --onedir --windowed --add-data="nav;nav" --collect-all=matplotlib app/gui_gnss.py
```

### 查看打包日志
打包过程中的详细日志会显示在控制台，如有错误可以根据日志排查。
