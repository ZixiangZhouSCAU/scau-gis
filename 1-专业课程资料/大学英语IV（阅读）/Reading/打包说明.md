# 英语阅读刷题软件 - 打包说明

## 🚀 快速打包

### 方法一：使用自动打包脚本（推荐）

1. **运行打包脚本**
   ```bash
   python build_exe.py
   ```

2. **等待打包完成**
   - 脚本会自动安装PyInstaller
   - 清理之前的构建文件
   - 打包生成exe文件

3. **查找exe文件**
   - 打包完成后，exe文件位于 `dist/英语阅读刷题软件.exe`

### 方法二：手动打包

1. **安装PyInstaller**
   ```bash
   pip install pyinstaller
   ```

2. **执行打包命令**
   ```bash
   pyinstaller --onefile --windowed --name=英语阅读刷题软件 test.py
   ```

## 📋 打包参数说明

| 参数 | 说明 |
|------|------|
| `--onefile` | 打包成单个exe文件 |
| `--windowed` | 隐藏控制台窗口 |
| `--name` | 指定exe文件名 |
| `--icon` | 指定图标文件 |
| `--add-data` | 包含额外文件 |
| `--optimize=2` | 代码优化级别 |
| `--strip` | 去除调试信息，减小文件大小 |

## 🎨 自定义图标

1. **准备图标文件**
   - 格式：`.ico`
   - 建议尺寸：256x256或128x128
   - 在线转换：[favicon-generator.org](https://www.favicon-generator.org/)

2. **使用图标**
   - 将 `icon.ico` 文件放在项目根目录
   - 重新运行打包脚本

## 🔧 优化建议

### 减小exe文件大小

1. **排除不需要的模块**
   ```bash
   --exclude-module=matplotlib,numpy,pandas,PIL
   ```

2. **使用UPX压缩**
   ```bash
   # 下载UPX: https://upx.github.io/
   pyinstaller --onefile --upx-dir=/path/to/upx test.py
   ```

### 提高启动速度

1. **使用--onedir模式**
   ```bash
   pyinstaller --onedir --windowed test.py
   ```
   - 优点：启动更快
   - 缺点：多个文件，不够便携

2. **预编译模块**
   - 删除不必要的导入
   - 使用懒加载

## 🐛 常见问题

### 问题1：打包失败
**解决方案：**
- 确保Python环境正常
- 更新PyInstaller：`pip install --upgrade pyinstaller`
- 检查文件路径是否正确

### 问题2：exe文件太大
**解决方案：**
- 使用虚拟环境，只安装必要包
- 添加更多exclude-module参数
- 考虑使用--onedir模式

### 问题3：exe运行报错
**解决方案：**
- 检查是否包含了所有必要文件
- 使用--console模式查看详细错误
- 确保目标机器有必要的运行库

### 问题4：杀毒软件误报
**解决方案：**
- 添加文件到杀毒软件白名单
- 使用--clean选项重新打包
- 考虑数字签名（需要证书）
